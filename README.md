## thegriefgang.com

A portfolio website for podcast The Grief Gang, hosted by Amber Jeffrey dealing with all things related to bereavement / grieving the passing away of loved ones

## Tech Stack

- React 
- Typescript
- NextJs
- GraphQL
- Contentful
- Styled Components
- Netlify
